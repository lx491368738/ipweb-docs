
## 接口概述

本接口用于查询指定服务器的当前带宽使用情况（近五分钟数据）。接口通过调用ZOSOC第三方API获取实时带宽数据，并返回格式化的结果。

> **注意：** 本接口需要有效的Token进行身份验证，未授权访问将被拒绝。

## 接口信息

**GET** `http://user.ipweb.cc/prod-api/open/customized/getCurBandwidth`

## 请求参数

### 请求头参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| token | String | **是** | 访问令牌，用于身份验证 |

## 响应格式

### 成功响应

**HTTP状态码：** 200 OK

```json
{
  "code": 200,
  "msg": "获取带宽信息成功",
  "data": [
    {
      "ip": "45.197.7.70",
      "curBandwidth": 125.50
    },
    {
      "ip": "45.197.7.114",
      "curBandwidth": 89.25
    }
  ]
}
```

### 响应字段说明

| 字段名 | 类型 | 说明 |
|--------|------|------|
| code | Integer | 响应状态码，200表示成功 |
| msg | String | 响应消息 |
| data | Array | 带宽数据数组 |
| data[].ip | String | 服务器IP地址 |
| data[].curBandwidth | BigDecimal | 当前带宽使用量（Mbps） |

### 错误响应

**HTTP状态码：** 200 OK（业务错误通过响应体中的code字段标识）

#### Token验证失败

```json
{
  "code": 500,
  "msg": "Token校验失败，访问被拒绝",
  "data": null
}
```

#### 服务异常

```json
{
  "code": 500,
  "msg": "获取带宽信息失败，请稍后重试",
  "data": null
}
```

```json
{
  "code": 500,
  "msg": "服务暂时不可用，请稍后重试",
  "data": null
}
```

## 调用示例

### cURL示例

```bash
curl -X GET "http://user.ipweb.cc/prod-api/open/customized/getCurBandwidth" \
     -H "token: your-access-token"
```

### JavaScript示例

```javascript
fetch('http://user.ipweb.cc/prod-api/open/customized/getCurBandwidth', {
    method: 'GET',
    headers: {
        'token': 'your-access-token'
    }
})
.then(response => response.json())
.then(data => {
    if (data.code === 200) {
        console.log('带宽数据:', data.data);
        data.data.forEach(item => {
            console.log(`IP: ${item.ip}, 带宽: ${item.curBandwidth} Mbps`);
        });
    } else {
        console.error('获取失败:', data.msg);
    }
})
.catch(error => {
    console.error('请求异常:', error);
});
```

### Java示例

```java
RestTemplate restTemplate = new RestTemplate();
HttpHeaders headers = new HttpHeaders();
headers.set("token", "your-access-token");

HttpEntity<String> entity = new HttpEntity<>(headers);

try {
    ResponseEntity<String> response = restTemplate.exchange(
        "http://user.ipweb.cc/prod-api/open/customized/getCurBandwidth",
        HttpMethod.GET,
        entity,
        String.class
    );
    
    System.out.println("响应结果: " + response.getBody());
} catch (Exception e) {
    System.err.println("请求异常: " + e.getMessage());
}
```

## 注意事项

- **Token安全：** 请妥善保管访问Token，避免泄露
- **请求频率：** 建议控制请求频率，避免对第三方API造成压力
- **数据时效性：** 返回的带宽数据为近五分钟的统计数据
- **错误处理：** 请根据响应中的code和msg字段进行相应的错误处理
- **IP范围：** 当前支持查询的IP地址为：45.197.7.70 和 45.197.7.114

## 错误码说明

| 错误码 | 错误信息 | 说明 |
|--------|----------|------|
| 200 | 获取带宽信息成功 | 请求成功 |
| 500 | Token校验失败，访问被拒绝 | Token无效或缺失 |
| 500 | 获取带宽信息失败，请稍后重试 | 第三方API调用失败 |
| 500 | 服务暂时不可用，请稍后重试 | 系统异常 |
